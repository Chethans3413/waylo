# Waylo — Travel Like Local 🌿

This is the official static website repository for **Waylo**, a travel experiences, treks, and local walks platform. The site is built using high-performance static HTML, CSS, and vanilla JavaScript.

## 🚀 Features

- **Responsive Landing Page:** Fully optimized for mobile, tablet, and desktop viewports.
- **Micro-Animations & Reveal Effects:** Engaging scroll reveal effects and clean SVG transitions.
- **Custom Assets:** Curated visual design using optimized assets (unused layout assets have been removed).
- **Static Hosting Ready:** Can be hosted directly on any static web hosting, CDN, or Hostinger Git deployment.

## 🛠️ Deployment via Hostinger Git

To deploy this static website directly to your main domain using Hostinger Git integration:

1. Log into your **Hostinger hPanel**.
2. Go to **Advanced** > **Git** on the left menu.
3. Set the repository settings:
   - **Repository Address:** `https://github.com/Chethans3413/waylo`
   - **Branch:** `main`
   - **Install Directory:** `public_html` *(Keep this as default to load directly from the root domain)*
4. Click **Create** (if you have an existing Git connection, you can click **Delete** or edit it to change the install directory to `public_html`).
5. Click **Deploy** to load the site files onto your live server.
